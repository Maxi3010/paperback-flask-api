# Paperback Flask API

Newspaper issue and subscription management API built with Flask and Flask-RESTX.

The project source is in the [`Assignment1`](./Assignment1) folder.

## Quick Start

```bash
cd Assignment1
python -m pip install -r requirements.txt
python start.py
```

Then open http://127.0.0.1:7890/ to use the Swagger UI.

## Tests

```bash
cd Assignment1
python -m pytest
```
